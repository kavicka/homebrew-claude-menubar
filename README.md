# homebrew-claude-menubar

Homebrew tap for [ClaudeMenuBar](https://github.com/kavicka/claude-menubar) — a
macOS menu bar app showing your running Claude Code chats.

```bash
brew install kavicka/claude-menubar/claude-menubar
claude-menubar setup
```

Requires Xcode Command Line Tools (`xcode-select --install`); the formula builds
from source, so there are no Gatekeeper warnings.

See the [main repo](https://github.com/kavicka/claude-menubar) for details.
